# VOXLINE JUCE UI Asset Pack

Generated from `VOXLINE_REBUILD_UI(1).md`.

Important:
- Do **not** use these as one pasted plugin screenshot.
- JUCE should draw panels, meters, buttons, knob arcs, labels, and dynamic values with C++ draw calls.
- Use these assets for optional logo marks, small icons, soft texture/glow overlays, or visual references.

Folders:
- `svg/`: scalable icon/logo/texture assets.
- `png/`: knob visual references / optional decorative parts.
- `preview/`: Light and Dark preview sheets.

Suggested path: `Source/UI/Assets/` and add to BinaryData / CMake `juce_add_binary_data`.
